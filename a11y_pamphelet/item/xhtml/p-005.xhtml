認證與檢驗

合規宣告

在規格中，提供了詮釋資料規格，讓EPUB可以經由製作者或者第三方認證，確保符合WCAG的哪一等級（A等級或者AA等級）；以及是否合乎EPUB可及性規格（目前為1.1版）。可以在metadata中以meta元素特性dcterms:conformsTo來宣告符合哪一等級的規格。

範例為：


<package …>
   <metadata>
      …
      <meta 
          property="dcterms:conformsTo"
          id="conf">
         EPUB Accessibility 1.1 - WCAG 2.1 Level AA
      </meta>
      …
   </metadata>
   …
</package>

符合的規格如下：

  • EPUB Accessibility 1.1 - WCAG 2.0 Level A

  • EPUB Accessibility 1.1 - WCAG 2.0 Level AA

  • EPUB Accessibility 1.1 - WCAG 2.0 Level AAA

  • EPUB Accessibility 1.1 - WCAG 2.1 Level A

  • EPUB Accessibility 1.1 - WCAG 2.1 Level AA

  • EPUB Accessibility 1.1 - WCAG 2.1 Level AAA

  • EPUB Accessibility 1.1 - WCAG 2.2 Level A

  • EPUB Accessibility 1.1 - WCAG 2.2 Level AA

  • EPUB Accessibility 1.1 - WCAG 2.2 Level AAA

認證者資訊

同時可以透過meta元素特性a11y:certifiedBy加入由誰認證。

範例為：


<metadata>
  …
  <meta
      property="dcterms:conformsTo"
      id="conf">
     EPUB Accessibility 1.1 - WCAG 2.1 Level AA
  </meta>
  <meta
      property="a11y:certifiedBy"
      refines="#conf">
     評量者或者組織的名字
  </meta>
  …
</metadata>

以及認證日期，需使用id來進行連結，範例為：


<meta
    property="a11y:certifiedBy"
    id="certifier">
   評量者或者組織的名字
</meta>
<meta
    property="dcterms:date"
    refines="#certifier">
   2021-09-07
</meta>

若該組織另提供認證標章，可以使用a11y:certifierCredential來產生關聯，範例為：


<meta
    property="dcterms:conformsTo"
    id="conf">
   EPUB Accessibility 1.1 - WCAG 2.1 Level AA
</meta>
<meta
    property="a11y:certifiedBy"
    refines="#conf"
    id="certifier">
   評量者或者組織的名字
</meta>
<meta
    property="a11y:certifierCredential"
    refines="#certifier">
   台灣電子書無障礙標章AA+
</meta>

若在網站上有提供檢查報告，可以透過a11y:certifierReport來提供連結，範例為：


<meta
    property="dcterms:conformsTo"
    id="conf">
   EPUB Accessibility 1.1 - WCAG 2.1 Level AA
</meta>
<meta
    property="a11y:certifiedBy"
    refines="#conf"
    id="certifier">
   評量者或者組織的名字
</meta>
<link
    rel="a11y:certifierReport"
    refines="#certifier"
    href="http://www.dpublishing.org.tw/a11y/report/1"/>
