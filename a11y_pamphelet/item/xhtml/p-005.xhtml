認證與檢驗

合規宣告

在規格中，提供了詮釋資料規格，讓EPUB可以經由製作者或者第三方認證，確保符合WCAG的哪一等級（A等級或者AA等級）；以及是否合乎EPUB無障礙輔助性規格（目前為1.1版）。可以在metadata中以meta元素特性dcterms:conformsTo來宣告符合哪一等級的規格。

範例為：


<package …>
   <metadata>
      …
      <meta 
          property="dcterms:conformsTo"
          id="conf">
         EPUB-A11Y-11_WCAG-21-AA
      </meta>
      …
   </metadata>
   …
</package>

符合的規格如下：

  • EPUB-A11Y-11_WCAG-20-A

  • EPUB-A11Y-11_WCAG-20-AA

  • EPUB-A11Y-11_WCAG-20-AAA

  • EPUB-A11Y-11_WCAG-21-A

  • EPUB-A11Y-11_WCAG-21-AA

  • EPUB-A11Y-11_WCAG-21-AAA

認證者資訊

同時可以透過meta元素特性a11y:certifiedBy加入由誰認證。

範例為：


<metadata>
  …
  <meta
      property="dcterms:conformsTo"
      id="conf">
     EPUB-A11Y-11_WCAG-21-AA
  </meta>
  <meta
      property="a11y:certifiedBy"
      refines="#conf">
     Acme Publishing Inc.
  </meta>
  …
</metadata>

以及認證日期，需使用id來進行連結，範例為：


<meta
    property="a11y:certifiedBy"
    id="certifier">
   EPUB Accessibility Evaluator
</meta>
<meta
    property="dcterms:date"
    refines="#certifier">
   2021-09-07
</meta>

若提供認證標章或者數位憑證，可以使用a11y:certifierCredential來產生關聯，範例為：


<meta
    property="dcterms:conformsTo"
    id="conf">
   EPUB-A11Y-11_WCAG-21-AA
</meta>
<meta
    property="a11y:certifiedBy"
    refines="#conf"
    id="certifier">
   EPUB Accessibility Evaluator
</meta>
<meta
    property="a11y:certifierCredential"
    refines="#certifier">
   A+ Accessibility Rating
</meta>

若在網站上有提供檢查報告，可以透過a11y:certifierReport來提供連結，範例為：


<meta
    property="dcterms:conformsTo"
    id="conf">
   EPUB-A11Y-11_WCAG-21-AA
</meta>
<meta
    property="a11y:certifiedBy"
    refines="#conf"
    id="certifier">
   EPUB Accessibility Evaluator
</meta>
<link
    rel="a11y:certifierReport"
    refines="#certifier"
    href="http://www.example.com/a11y/report/9780000000001"/>
